###  Roller version 1.6.4-alpha-release Güncelleme


###  1. Sistemi durdurun

```
roller rollapp services stop
```

###  2- yeni sürümü yükleyin

```
curl https://raw.githubusercontent.com/dymensionxyz/roller/main/install.sh | bash 
```

###  3- versiona bakın

```
roller version 
```
